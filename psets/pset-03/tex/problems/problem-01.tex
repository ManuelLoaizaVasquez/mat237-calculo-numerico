  Determine si $f$ es un spline c\'ubico con nudos $-1$, $0$, $1$ y $2$.
  \[
    f(x) = 
    \begin{cases}
      1 + 2(x + 1) + (x + 1)^3 & -1 \leq x < 0, \\
      3 + 5x + 3x^2 & 0 \leq x < 1, \\
      11 + (x - 1) + 3 (x - 1)^2 + (x - 1)^3 & 1 \leq x \leq 2.
    \end{cases}
  \]
\end{statement}
